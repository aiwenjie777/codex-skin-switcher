# Caishen Readable

Public-safe macOS skin package for Codex Skin.

## Install

```bash
macos/scripts/apply-skin-package-macos.sh --package macos/skin-packages/caishen-readable
```

Then switch later with:

```bash
macos/scripts/switch-theme-macos.sh --id caishen-readable
```

## Notes

- Data-only package: no scripts, binaries, private screenshots, or workspace content.
- Light fortune artwork with warm panels and higher-contrast text tokens.
- Source project: https://github.com/ChannelerH/codex-skin-packs
