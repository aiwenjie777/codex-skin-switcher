# Codex Theme Creator

- Produce a concrete theme folder, not just a design proposal.
- Use lowercase hyphen-case directory names.
- Keep backgrounds coordinated and preserve readable native controls.
- Do not modify Codex `config.toml`, the official app, `app.asar`, or signatures.
- For Windows themes, include `dream-reference.png`, `dream-skin.css`, and optional `effects.css`, `sidebar.png`, `main.png`, and `polaroid.png`.
- For macOS distributable skins, follow [`docs/SKIN_PACKAGES.md`](../../docs/SKIN_PACKAGES.md).
